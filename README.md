# ESP32 WiFi Demonstration

Following an ESP-IDF WiFi demonstration to get up and running


## Reference Example

- [Wi-Fi Station Example - Getting Started](https://github.com/espressif/esp-idf/blob/master/examples/wifi/getting_started/station/README.md)

# Steps

Starts with .ESP-IDF 'Wi-Fi Station Example'

1. Open Demo in VS Code [1]


Configure the project

	idf.py menuconfig

2. Build and Flash

3. Example Output

4. Ping Website

5. Receive Ping

6. Respond to Ping 'r1'
